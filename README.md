# GitHub Index Page

A simple, modern index page that lists public GitHub repositories for **@asadali2004**. It fetches repos from the GitHub API, filters out forks, and supports search.

## Features
- Recent public repos (sorted by last update)
- Search by name or description
- Repo stats (stars, forks, last updated)
- Clean, responsive UI

## How to use
1. Open `index.html` in your browser.
2. The page will load repositories automatically.

## Customize
- Change the username in `index.html`:
  ```js
  const username = "asadali2004";
  ```

## Notes
- GitHub API has rate limits for unauthenticated requests.
